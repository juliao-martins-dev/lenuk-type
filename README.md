# lenuk-type

AI-assisted typing game built with Next.js App Router and TypeScript.

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`.
